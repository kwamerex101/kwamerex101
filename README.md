## 🏆 Highlighted Work

- **[SEO Audit Studio](https://github.com/kwamerex101/seo-audit-studio):** multi-company SEO + GEO (Generative Engine Optimization) audit app. Crawls a site, scores it deterministically and with AI, then lets you chat with the report. Nuxt + TypeScript.
- **[Firestore Query Studio](https://github.com/kwamerex101/firestore_query_studio):** local-first Electron app that queries Google Cloud Firestore in plain English. The LLM output becomes a Zod-validated plan, executed read-only via the Admin SDK or emulator. Bring your own key.
- **[phone_screen_mirror](https://github.com/kwamerex101/phone_screen_mirror):** mirror and control a USB-connected iPhone from macOS while the phone stays usable. Ships with an MCP server for automation.
- **[bmoni_date_picker](https://github.com/kwamerex101/bmoni_date_picker):** a Cupertino-styled date picker for Flutter, extracted as a reusable shared component.
- **[Grammatically Correct](https://github.com/kwamerex101/grammatically_correct):** a native macOS writing and grammar helper, built in Swift.
- **[Listen To Me](https://github.com/kwamerex101/listen_to_me):** a native macOS app built in Swift.
